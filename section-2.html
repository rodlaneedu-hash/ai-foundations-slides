How Generative AI Works

Six building blocks. Each one paired with the misconception it dissolves.

1
Building Block

Training data — patterns

Generative AI is trained on massive amounts of text — books, websites, articles, code.

During training, it learns patterns: which words and ideas tend to appear together, in what order, in what situations.

Misconception dissolved
AI has been programmed with knowledge and facts.

It hasn't. It has patterns. No one sat down and taught it that Wellington is the capital of New Zealand — it encountered that pattern frequently enough in training data to reproduce it.

2
Building Block

Token prediction

When you give it a prompt, it generates a response one word at a time, each time asking: "Given everything so far, what word should come next?"

It does this thousands of times to produce a single response.

Misconception dissolved
AI understands what it's saying.

It doesn't. It's predicting what comes next based on patterns. The output reads as if it understands because human language follows patterns — and the model is very good at reproducing them. But there is no comprehension behind it.

3
Building Block

It doesn't give the same answer twice

There's a degree of randomness in how it picks the next word. The same prompt can produce different outputs each time.

It's not retrieving a stored answer — it's generating a new one every time.

Misconception dissolved
There's a right answer in there if I ask the right way.

There isn't. Every response is freshly generated. This is also why two people can get different answers to the same question.

4
Building Block

No memory. No knowledge. No reasoning.

No access to the internet (unless given tools). No memory between conversations (unless designed to). Cannot verify its own claims.

It produces text that looks like reasoning because humans reason using patterns of language — and the AI is very good at copying those patterns.

Misconception dissolved
AI knows things. AI is always right.

It will state a falsehood with the same confidence as a fact, because sounding confident is just another pattern it learned. This is called hallucination — and it won't go away. It's inherent to how the technology works.

5
Building Block

It's not a search engine

Google searches an index of web pages and shows you links to information that already exists.

A generative AI tool constructs an answer that sounds right, rather than retrieving a specific page. If you ask for references, it may invent convincing ones that don't exist.

Misconception dissolved
I asked AI, and it said...

That is not the same as "I looked it up and found..." Treating AI output as a retrieved fact is the single fastest way to get burned. If you need a fact, go to the source.

6
Building Block & Misconception

Its knowledge has a use-by date

Every model was trained on data up to a specific point — its knowledge cutoff. It doesn't know what happened after that date.

If you ask about recent events, it will either say it doesn't know — or more dangerously, generate a plausible-sounding answer based on older patterns.

A human must always be in the loop.

Generative AI works by predicting patterns. That's all it does. It does not know, understand, reason, or verify.

Every strength and every risk flows from this single fact.

Checking. Interpreting. Every time.
